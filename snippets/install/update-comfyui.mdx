## How to Update Hanzo Desktop

Currently, Hanzo Desktop updates use automatic detection updates, please ensure that automatic updates are enabled in the settings

![Hanzo Desktop Settings](/images/desktop/hanzo-studio-desktop-update-setting.jpg)

You can also choose to manually check for available updates in the `Menu` --> `Help` --> `Check for Updates`

![Hanzo Desktop Check for Updates](/images/desktop/desktop_check_for_updates.jpg)
