Hanzo Studio 用户目录是存储您的个人设置、工作流和自定义内容的地方。位置取决于您的安装类型：

<Tabs>
<Tab title="桌面版Windows">
```
C:\Users\<your username>\AppData\Roaming\Hanzo Studio
```
</Tab>
<Tab title="桌面版macOS">
```
~/Library/Application Support/Hanzo Studio
```
</Tab>
<Tab title="桌面版Linux">
```
~/.config/Hanzo Studio
```
</Tab>
<Tab title="手动安装">
用户目录位于您的 Hanzo Studio 安装文件夹中：
```
<Hanzo Studio 安装路径>/user/<用户名>
```

例如，如果您将 Hanzo Studio 克隆到 `C:\Hanzo Studio`，您的用户目录将是 `C:\Hanzo Studio\user\default`（或者如果您设置了自定义用户名，则为 `C:\Hanzo Studio\user\john`）。

<Note>
Hanzo Studio 支持每个安装支持多个用户。如果您没有配置自定义用户名，默认为 "default"。每个用户在 `user` 文件夹内都有自己的子目录。
</Note>
</Tab>
<Tab title="便携版">
用户目录位于您的 Hanzo Studio 便携版文件夹中：
```
<HanzoStudio_windows_portable>/Hanzo Studio/user/<用户名>
```

例如：`HanzoStudio_windows_portable/Hanzo Studio/user/default`

<Note>
Hanzo Studio 支持每个安装支持多个用户。如果您没有配置自定义用户名，默认为 "default"。每个用户在 `user` 文件夹内都有自己的子目录。
</Note>
</Tab>
</Tabs>

此位置包含您的工作流、设置和其他用户特定文件。
