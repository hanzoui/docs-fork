Built-in nodes are Hanzo Studio's default nodes. They are core functionalities of Hanzo Studio that you can use without installing any third-party custom node packages.

## About built-in node document

We have now added built-in node help documentation, so the content of this section is periodically synced from [this repo](https://github.com/hanzoui/embedded-docs). We will update the content manually once a week.
