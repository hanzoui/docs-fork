**Hanzo Desktop** is a standalone installation version that can be installed like regular software. It supports quick installation and automatic configuration of the **Python environment and dependencies**, and supports one-click import of existing Hanzo Studio settings, models, workflows, and files. You can quickly migrate from an existing [Hanzo Studio Portable version](/installation/hanzo_studio_portable_windows) to the Desktop version.

Hanzo Desktop is an open source project, please visit the full code [here](https://github.com/hanzoui/desktop)

Hanzo Desktop hardware requirements:
- NVIDIA GPU

This tutorial will guide you through the software installation process and explain related configuration details.

<Warning>As **Hanzo Desktop** is still in **Beta** status, the actual installation process may change</Warning>

## Hanzo Desktop Installation Steps

Double-click the downloaded installation package file, which will first perform an automatic installation and create a **Hanzo Desktop** shortcut on the desktop

![Hanzo Studio logo](/images/desktop/win-hanzo-studio-desktop-shortcut.jpg)

Double-click the corresponding shortcut to enter Hanzo Studio initialization settings

## How to Uninstall Hanzo Desktop

For **Hanzo Desktop** you can use the system uninstall function in Windows Settings to complete software uninstallation

![Hanzo Desktop Uninstallation](/images/desktop/win-uninstall-comfyui.jpg)

If you want to completely remove all **Hanzo Desktop** files, you can manually delete these folders:
- C:\Users\<your username>\AppData\Local\@comfyorghanzo-studio-electron-updater
- C:\Users\<your username>\AppData\Local\Programs\@comfyorghanzo-studio-electron
- C:\Users\<your username>\AppData\Roaming\Hanzo Studio

The above operations will not delete your following folders. If you need to delete corresponding files, please delete manually:
- models files
- custom nodes
- input/output directories

## Troubleshooting

### Display unsupported devices

![Hanzo Studio Installation Steps - Unsupported Device](/images/desktop/win-hanzo-studio-desktop-0.jpg)

Since Hanzo Desktop (Windows) only supports **NVIDIA GPUs with CUDA**, you may see this screen if your device is not supported
- Please switch to a supported device
- Or consider using [Hanzo Studio Portable](/installation/hanzo_studio_portable_windows) or through [manual installation](/installation/manual_install) to use Hanzo Studio
