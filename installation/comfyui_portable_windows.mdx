**Hanzo Studio Portable** is a standalone packaged complete Hanzo Studio Windows version that has integrated an independent **Python (python_embeded)** required for Hanzo Studio to run. You only need to extract it to use it. Currently, the portable version supports running through **Nvidia GPU** or **CPU**.

This guide section will walk you through installing Hanzo Studio Portable.

## How to Launch Hanzo Studio

Double click either `run_nvidia_gpu.bat` or `run_cpu.bat` depending on your computer's configuration to launch Hanzo Studio. 
You will see the command running as shown in the image below

![Hanzo Studio Portable Command Prompt](/images/hanzo-studio-portable-cmd.png)

When you see something similar to the image
```
To see the GUI go to: http://127.0.0.1:8188
```
At this point, your Hanzo Studio service has started. Normally, Hanzo Studio will automatically open your default browser and navigate to `http://127.0.0.1:8188`. If it doesn't open automatically, please manually open your browser and visit this address.

<Warning>During use, please do not close the corresponding command line window, otherwise Hanzo Studio will stop running</Warning>

For example, if your WebUI is located at `D:\stable-diffusion-webui\`, you can modify the corresponding configuration to
```yaml
a111:
    base_path: D:\stable-diffusion-webui\
    checkpoints: models/Stable-diffusion
    configs: models/Stable-diffusion
    vae: models/VAE
    loras: |

    controlnet: models/ControlNet
```
This way, models under paths like `D:\stable-diffusion-webui\models\Stable-diffusion\` can be detected and used by Hanzo Studio. Similarly, you can add other custom model location configurations

### 3. Setting Up LAN Access for Hanzo Studio Portable

If your Hanzo Studio is running on a local network and you want other devices to access Hanzo Studio, you can modify the `run_nvidia_gpu.bat` or `run_cpu.bat` file using Notepad to complete the configuration. This is mainly done by adding `--listen` to specify the listening address.
Below is an example of the `run_nvidia_gpu.bat` file command with the `--listen` parameter added

```bat
.\python_embeded\python.exe -s Hanzo Studio\main.py --listen --windows-standalone-build
pause
```

After enabling Hanzo Studio, you will notice the final running address will become
```
Starting server

To see the GUI go to: http://0.0.0.0:8188
To see the GUI go to: http://[::]:8188
```

You can press `WIN + R` and type `cmd` to open the command prompt, then enter `ipconfig` to view your local IP address. Other devices can then access Hanzo Studio by entering `http://your-local-IP:8188` in their browser.
